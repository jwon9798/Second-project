# ClipQuiz

The global guessing game platform — a Machugi.io-inspired international quiz experience.

## Quick Start

```bash
npm install
npm run dev
```

## Deploy on Vercel (important)

1. Connect repo: https://github.com/jwon9798/Second-project
2. **Root Directory** must be empty / `.` (NOT `clipquiz`)
3. Framework: **Next.js** (auto-detected)
4. Click **Redeploy** after pulling latest `main`

Your site URL should show ClipQuiz at `/en`.

## Features

- Image / Crop / Audio quiz modes
- Percentile ranking
- UGC quiz creation
- 5 languages: EN, KO, JA, ES, PT
